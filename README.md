# Cafe Table

A Minecraft 1.20.1 Forge mod for the ModestCraft / InfrastructureSickos modpack.

## Description

Adds a **Cafe Table** block that attracts nearby villagers to socialize around it. Villagers seated at a cafe table will occasionally trade food items for emeralds, creating a dynamic village economy feel.

## Features

- Cafe Table block that acts as a social gathering point for villagers
- Villagers path-find to nearby tables when idle
- Food-for-emeralds trade interactions at the table

## Requirements

- Minecraft 1.20.1
- Forge 47.3.0+
- Java 17

## Building

```bash
./gradlew build
```

Output jar will be in `build/libs/`.
